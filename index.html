MQTT

SYNOPSIS

MQTT is an embeddable version of EMQ 2.1.2, the lightweight MQTT 5.0 server for Erlang by Feng Lee.

RELEASE NOTES

  • MQTT 7.0 13 FEB 2018

USAGE

EMQ is available as one of rebar3 dependencies: emqttd and emq_dashboard. The first is server only instance and the latter is server alogin with dashboard. It can be accessed after start of emq_dashboard application at localhost:18083 by using admin:skynet credentials.
